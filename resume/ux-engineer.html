인터랙션과 접근성을 시스템으로 설계하는 데 집중합니다.

5년차 개발자로 React와 TypeScript 진영에서 서비스의 설계, 배포, 운영 전반을 담당하고 있습니다.
현재는 째깍악어에서 FE팀을 리딩하며, 국내 대표 돌봄 플랫폼 째깍악어 하이브리드 앱을 담당하고 있습니다.

a11y과 인클루시브 디자인에 대한 관심과 전문성을 바탕으로 디자인 시스템과 인터랙션을 일관된 구조로 설계하고 서비스 전반의 UX 품질을 개선합니다.

멀티모달(탈시각 UI)을 기반으로 한 접근성 라이브러리 @uturi/sonification의 메인테이너로 활동하고 있습니다.
최근에는 자체 물리 엔진을 활용한 모바일 인터랙션 라이브러리 @maru-ui를 개발하며, 인터랙션을 재사용 가능한 구조로 추상화하는 작업을 진행하고 있습니다.

더 자세한 프로젝트와 개발 철학은포트폴리오에서 확인하실 수 있습니다.

프로젝트

멀티모달 라이브러리 @uturi/sonification 운영

배경 및 접근

  • 시계열 차트, 테이블 등 시각 중심 UI는 비시각 사용자에게 정보 접근성이 크게 제한되는 문제 존재
  • 특히 데이터의 흐름과 변화량은 스크린 리더만으로 직관적으로 인지하기 어려움
  • 데이터를 보는 대신 들을 수는 없을까라는 문제의식에서 프로젝트 시작

작업

  • 수치 데이터를 frequency(주파수), volume(크기), rhythm(리듬) 등 다양한 청각 요소로 매핑하는 데이터 음향화(sonification) 모델 설계 및 구현
  • 단순 주파수 기반 표현의 한계를 개선하기 위해 7음계 기반 melody 모드 도입하여 데이터 변화량 인지성 향상
  • 다양한 WaveForm 형태를 추가 지원하여 데이터 특성에 따른 오디오 표현 선택 가능성 제공 및 데이터의 포괄성 개선
  • 수백~수천 개 데이터 처리 시 발생하는 메인 스레드 블로킹 문제 해결을 위해 Web Worker 기반 오디오 생성 파이프라인 구축
  • Framework-Agnostic 라이브러리 지향, 멀티 프레임워크 지원(React, Vue, Svelte, Vanilla.js) 및 타입 안전성 확보
  • 다양한 modality로 확장 고려하여, TurboRepo 기반 모노레포로 운영

결과

  • 현재 v2.2.1, NPM 기준 다운로드 수 약 1.5k+, 오픈소스 운영 및 지속적인 기능 개선 진행
  • Web Vibration API를 활용한 @uturi/haptification 확장 진행 중
  • 관련 링크: 라이브러리 문서(데모 포함), 라이브러리 개발 기록, 음성 출력 통합 테스트 구축 기록
